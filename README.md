# Pyramid
<h3>Implemention of OpenGL assignment (Pyramid rendered) </h3>

<h4>Commands to initialise git module :</h4>
<br>

git submodule init <br>
git submodule update <br>

<h4>Now, build the program using following commands:</h4>

cmake -S  .  -B ./build <br>
cd build <br>
make .j12 <br>
./main   <br>
